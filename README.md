# Praise & Pay Portal

A web application for sending praise and appreciation messages with different support levels.

## Features

- **Main Portal**: Send praise messages with different support levels (Basic, Premium, Elite)
- **Admin Portal**: View all praises, track earnings, and export data
- **Secure Login**: Password-protected admin access with change password functionality
- **Responsive Design**: Works on desktop and mobile devices
- **Data Persistence**: Uses localStorage to save praises and admin settings

## Praise Levels

- 💝 **Basic Praise**: $5 - A simple thank you message
- ⭐ **Premium Praise**: $15 - Detailed appreciation with emoji
- 👑 **Elite Praise**: $50 - Ultimate praise experience

## Admin Features

- View all submitted praises
- Track total earnings and statistics
- Export data to CSV format
- Change admin password
- Real-time statistics dashboard

## Default Login

- Username: `admin`
- Password: `admin123`

## Technology Stack

- HTML5
- CSS3 with modern styling
- Vanilla JavaScript
- LocalStorage for data persistence
- Responsive design with mobile support

## How to Use

1. **Sending Praise**:
   - Select a praise level
   - Enter your name and message
   - Click "Pay & Submit Praise"

2. **Admin Access**:
   - Click "Admin Portal" on the main page
   - Login with admin credentials
   - View statistics and all praises
   - Export data if needed

## Deployment

This application is designed to work with GitHub Pages. Simply push the files to a GitHub repository and enable GitHub Pages in the repository settings.
